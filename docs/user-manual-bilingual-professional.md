## 1. Product Overview
### English
Construction ERP is a company-scoped web platform for daily construction operations, commercial controls, and finance workflows. It combines dispatch, reports, masters, parties, vehicles, and accounts in one controlled system.

### हिंदी
Construction ERP एक company-scoped web platform है जिसमें daily operations, commercial controls और finance workflows एक ही system में manage होते हैं। Dispatch, reports, masters, parties, vehicles और accounts एक integrated flow में काम करते हैं।

## 2. Login and Session Basics
### English
1. Open company login URL.
2. Enter login ID and password.
3. Select company context if prompted.
4. If temporary password is forced, change password immediately.

Good practice:
- Always verify active company scope in the header before saving data.
- Logout after shift closure.

### हिंदी
1. Company login URL खोलें।
2. Login ID और password डालें।
3. Prompt आए तो company context चुनें।
4. Temporary password force हो तो तुरंत password बदलें।

Best practice:
- Data save करने से पहले header में active company scope check करें।
- Shift बंद होने के बाद logout करें।

## 5. Operations Flow (Practical)
### English
Core operational sequence:
1. Maintain masters (materials, shifts, units, etc.)
2. Maintain party and rate data
3. Create and control party orders
4. Record dispatch with billing details
5. Monitor project/crusher reports
6. Track exceptions and resolve backlog

### हिंदी

2. Party और rates update करें
3. Party orders create और control करें
4. Billing details के साथ dispatch record करें
5. Project/crusher reports monitor करें
6. Exceptions track करके backlog close करें

## 6. Accounts and Finance Usage (Critical)
### English
Accounts section includes:
- Accounts Dashboard
- Chart of Accounts and Ledgers
- Voucher Entry (maker-checker workflow)
- Receivables
- Payables
- Cash/Bank
- Finance Reports
- Period Controls
- Finance Policy Controls

Voucher workflow:
1. Create draft with balanced debit/credit lines
2. Submit for approval
3. Approve through checker role
4. Post after approval
5. Reverse (if correction required)

Period control:
- Closed periods block posting actions
- Reopen is controlled and audited

### हिंदी
Accounts section में शामिल है:
- Accounts Dashboard
- Chart of Accounts और Ledgers
- Voucher Entry (maker-checker workflow)
- Receivables
- Payables
- Cash/Bank
- Finance Reports
- Period Controls
- Finance Policy Controls

Voucher workflow:
1. Balanced debit/credit lines के साथ draft बनाएं
2. Approval के लिए submit करें
3. Checker role से approve करें
4. Approval के बाद post करें
5. Correction हो तो reverse करें

Period control:
- Closed period में posting block होती है
- Reopen controlled और audited action है

## 9. Common User Issues
### English
- Data not visible: Check filters and company scope.
- Action blocked: Check role permission and workflow status.
- Posting blocked: Verify period status and approval state.
- Login issue: Validate company code and credentials.

### हिंदी
- Data नहीं दिख रहा: Filters और company scope check करें।
- Action blocked: Role permission और workflow status check करें।
- Posting blocked: Period status और approval state check करें।
- Login issue: Company code और credentials verify करें।
